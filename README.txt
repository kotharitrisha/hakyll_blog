Contents of the repository:

1. site.hs ==> Main Haskell file for generating the blog
2. posts/ ==> Directory containing markdown files of all blog posts
3. links/ ==> Directory containing markdown files of all interesting links
4. css/ ==> All css templates
5. images/ ==> All image files
6. templates ==> Directory containing all templates: default, links, and posts
7. about.markdown/contact.markdown ==> Main files for these static pages
8. index.html ==> Template for the first page of the blog

When the site is built, a folder named _site is created with all the respective HTML files.
